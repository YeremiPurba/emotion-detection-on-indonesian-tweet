# Deteksi Emosi Tweet Menggunakan Support Vector Machine

---

Proyek ini melakukan deteksi emosi pada Tweet dengan menggunakan algoritma _Support Vector Machine_ dan metode ekstraksi fitur TF-IDF.
Jumlah emosi yang dideteksi ada sebanyak lima kelas, yakni marah, takut, senang, cinta, dan sedih. 

---

## Dataset
- **Indonesian Twitter Emotion Dataset**
- Sumber: https://www.kaggle.com/datasets/wahyuikbalmaulana/indonesian-twitter-emotion-dataset/data

---

## Setup
### 1. Clone the repository
```bash
git clone https://github.com/YeremiPurba/emotion-detection-on-indonesian-tweet.git
```

### 2. Import Database
Unduh file `emotion_detection.sql` yang terdapat di `emotionDetection/database/`  <br>
Import file ke dalam aplikasi DBMS anda.
